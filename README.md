# Flashcard Quiz App

A simple flashcard quiz app built with Python and Tkinter.

## Description

This app allows users to create flashcards, view saved flashcards, and take a quiz using a simple graphical user interface. Flashcards are saved in a JSON file so the data remains available after closing the app.

## Features

- Add flashcards
- View saved flashcards
- Take a quiz
- Check answers
- View final score
- Save and load flashcards using JSON
- Simple Tkinter GUI

## How to Run

Run the app with:

```bash
python main.py
